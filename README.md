# business-ai

Experiments in AI-built business software.

## [`ai-stores/`](ai-stores/README.md)

A single app instance that serves **many storefronts** at `stores.com/{store}`,
managed by **one shared admin** — add a store from the UI, no redeploy. Built on
[`mdb-engine`](ai-stores/blog.md) with per-request data scoping, dynamic store
provisioning, and a local-AI conversational editor.
